Connexion à l’API

Seules les personnes qui ont créé un compte utilisateur peuvent, en étant connectées, faire des actions sur la base de données.

Sur Postman, faire un POST de http://localhost:8080/auth/register

Créer un utilisateur

Faire un POST de http://localhost:8080/auth/login avec les informations nécessaires.

Se connecter à un compte utilisateur

Faites un GET de http://localhost:8080/catways en ajoutant le token dans l’onglet Authorization.

Créer un catway

Faites un POST de http://localhost:8080/catways/add avec les champs nécessaires.

Créer une réservation

Faites un POST de http://localhost:8080/catways/IDNUMBER/reservations avec les informations requises.
